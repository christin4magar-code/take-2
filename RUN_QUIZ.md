# 🚀 How to Run the Calculus Study Bot

To run the interactive quiz without installing anything on your computer, follow these steps:

1. **Launch Environment:** Click the "Launch Interactive Quiz" button on the main [README](./README.md).
2. **Wait for Load:** GitHub will set up a virtual "Codespace" for you (takes about 30 seconds).
3. **Execute Command:** In the black terminal box at the bottom of the screen, type:
   ```bash
